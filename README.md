Sticky Mark is a simple browser extension that provides a persistent sticky note. Users can type notes in one tab, switch to any other tab, and still see the same content when reopening the extension.
Features
Single sticky note accessible from the extension icon
Typed content is saved automatically
Notes remain available even after switching tabs
Simple and distraction-free interface
Lightweight and fast

Tech Stack
HTML
CSS
JavaScript
Browser Extension APIs
Local storage (for saving notes)
